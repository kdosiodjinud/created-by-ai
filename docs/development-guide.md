# Development Guide

## Local Setup
1. Requirements
   - PHP 8.1+
   - Composer

2. Installation Steps
   ```bash
   git clone https://github.com/kdosiodjinud/created-by-ai.git
   cd created-by-ai
   composer install
   ```

## Running Tests
```bash
composer test
```

## Code Style
Use PHPCodeSniffer:
```bash
composer lint
```
